# 중간고사 준비와 실습 경험

## 선택한 템플릿 및 주제
- Space_shooter.py / '적을 동료로 만들어 같이 싸우기.'

## 사용한 AI 프롬포트 (요약)
1. 코드 구조 설명 요청
- 게임은 정지 화면이 아니라 1초에 60번(60 FPS) 화면을 새로 그리는 과정입니다. -> screen.fill(GRAY)로 이전 그림을 싹 지우고, 캐릭터들의 바뀐 위치를 새로 그려서 우리 눈에는 움직이는 것처럼 보임.
- 화면의 왼쪽 위가 (0, 0) 지점입니다. -> 플레이어: 화살표 키를 누르면 X, Y 좌표가 변하며 이동함. / 총알: Y 좌표가 계속 줄어들며 위로 올라감. / 적: Y 좌표가 계속 늘어나며 아래로 내려옴.
- colliderect라는 기능을 사용하여 "총알 사각형과 적 사각형이 겹쳤는가?"를 매 순간 체크합니다. 겹쳤다면 점수를 올리고 화면에서 없애버립니다.
- 화면 밖으로 나간 총알이나 적은 눈에 안 보일 뿐 데이터상으로는 계속 존재합니다. 이를 방치하면 게임이 무거워집니다. -> if b.bottom > 0 같은 조건으로 화면 밖으로 나간 객체들을 리스트에서 수시로 삭제함.
- score // 50 공식을 사용하여 50점마다 레벨이 오르게 설계되어 있습니다. 레벨이 오를수록 적이 생성되는 간격은 짧아지고, 내려오는 속도는 빨라져 긴장감을 유도합니다.

2. 기획서 구체화 요청
- 질문: 아래는 내가 만들 게임의 기획서 초안이야. 빈칸을 채우고 핵심 규칙을 더 구체적으로 만들어줘. 난이도가 어떻게 올라가는지도 제안해줘. 중간고사 3주 안에 완성 가능한 규모인지도 알려줘.
- 답변: 동료 수집 시스템: 적 처치 시 10% 확률로 '구출 아이템' 드랍. 획득 시 랜덤하게 1성 동료가 플레이어 주변에 배치됨, 머지(Merge) 시스템: 별도의 조합 창이 아닌, 게임 플레이 중 같은 성급의 동료끼리 충돌하거나 근접하면 상위 성급으로 즉시 합쳐짐, 라운드가 올라갈수록 적의 수치만 높이는 것이 아니라, '패턴'을 추가해 긴장감을 줍니다, 성급 시스템: 최대 3성 혹은 5성까지 설정 (3주 완성형으로는 3성 추천). / 보스 특징: 1라운드마다 등장. / 아이템: '실드(방어)', '폭탄(화면 정리)' 아이템 추가.

3. 구현 가능성 확인
- 질문: 내가 기획한 게임에서 [차별점] 기능을 구현하려면 Pygame 코드에서 어떤 부분을 어떻게 수정해야 해? 수정이 어려운 부분이 있으면 미리 알려줘.
- 답변: 결론부터 말씀드리면: 가능합니다. 단, 이미지 리소스에 너무 힘을 빼지 않고 '기능 구현'에 집중한다는 조건하에 가능합니다. 
- [1주차] 기반 다지기 (Core)
기존 'Space Shooter' 템플릿 분석 및 정리.
플레이어 이동 및 기본 총알 발사.
적 생성 루프 및 라운드 전환 시스템(Score 기반) 구현.
- [2주차] 동료 및 합성 (Special)
적 처치 시 아이템 드랍 구현.
동료(Follower) 로직 구현 (플레이어 좌표를 일정 시간 차로 따라오게 설정).
핵심: 동료 간의 충돌 감지 및 성급 업그레이드 로직 완성.
- [3주차] 보스 및 폴리싱 (Final)
미니보스/최종보스 AI 및 탄막 패턴 구현.
메뉴 화면, 게임 오버/승리 화면 UI 제작.
사운드 효과 삽입 및 버그 수정.
-> 중간고사 프로젝트라면, '합성이 정상적으로 작동하는가'가 가장 큰 가산점 요인이 될 것입니다.

## AI 답변에서 도움이 된 것
- 난이도 설계 (라운드 시스템)라운드가 올라갈수록 적의 수치만 높이는 것이 아니라, '패턴'을 추가해 긴장감을 줍니다.
- Lv.1: 기본 잡몹, 일직선으로 내려오는 적
- Lv.2: 잡몹 + 미니보스, 적의 이동 속도 +20%, 미니보스가 조준 탄환 발사
- Lv.3: 패턴 잡몹 + 보스, 적들이 지그재그로 이동, 보스가 화면의 30%를 덮는 탄막 사용
- 무한: 엘리트 적 등장, 체력이 아주 높은 적이 등장하여 '합성된 동료'의 필요성 강조
-> AI가 명확한 수치와 기믹을 설정해 만들어 주는 것이 3주 안에 완성해야한다는 특성 상 굉장히 도움이 되었음. 특히 AI는 원하는 결과물을 내기 위해서는 모호하게 말하지 말고 명확하게, 그리고 수치로 설명해야 원하는대로 만들어질 확률이 올라감.

## AI 답변을 수정하거나 버린 것
- 게임 시스템 상세 (빈칸 채우기)
- 성급 시스템: 최대 3성 혹은 5성까지 설정 (3주 완성형으로는 3성 추천).
- 보스 특징: 5라운드마다 등장. 특정 체력 이하가 되면 광폭화(패턴 빨라짐).
- 아이템: 동료 구출 외에 '실드(방어)', '폭탄(화면 정리)' 아이템 추가.
-> 3주라는 제한시간과 큰틀을 바꾸지 못한다는 점에서 게임자체가 단순해질 수 밖에 없음. 단순한 게임에서 '동료 합성 시스템'으로 차별점을 줘야하는데 광폭화 패턴까지 만들 시간이 있을지 의문, 그리고 단순한 게임 특성상 쉽게 질리는데 5라운드마다 보스가 나오면 게이머가 쉽게 싫증을 느낄 수 있음. 1라운드 마다 보스를 출현 시키고, 보스를 단순히 수치 뻥튀기를 하는게 3주 안에 할 수 있는 최선으로 판단 됨. 또한 둉료 구출 아이템은 만들고 문제없이 적용시키는데 에로사항이 있을 것으로 보이며, 캐릭터가 적 처치만 해도 10%의 확률로 동료로 만들 수 있음. 실드(방어)와 폭탄(화면 정리)는 시간이 남을 때 추가 가능.
